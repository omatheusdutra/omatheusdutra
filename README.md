## 🚀 Projetos em destaque

- **[Email Smart Reply](https://github.com/omatheusdutra/desafio-autoU)**  
  Problema: classificar emails e gerar respostas prontas com rapidez e consistência.  
  Solução: FastAPI com zero-shot Transformers, heurísticas locais e opção de GPT; interface Jinja com upload único e lote via ZIP.  
  Impacto: pipeline completo, com testes (Pytest) e cobertura.

- **[Star Wars API](https://github.com/omatheusdutra/star-wars-api-gcp-holonet)**  
  Problema: oferecer uma experiência rica para fãs, unificando dados da SWAPI com filtros e ordenação.  
  Solução: API com cache, correlações e grafo; desenhada para GCP (Cloud Functions + API Gateway).  
  Impacto: estrutura pronta para escala e baixa latência em ambiente serverless.

- **[Analytics de Vendas](https://github.com/omatheusdutra/Teste_Analytics_MatheusDutra)**  
  Problema: transformar dados brutos em insights acionáveis sobre produtos e tendências.  
  Solução: simulação de vendas, limpeza de dados, análise exploratória e visualizações objetivas.  
  Impacto: informações estratégicas para tomada de decisão com foco em negócio.
